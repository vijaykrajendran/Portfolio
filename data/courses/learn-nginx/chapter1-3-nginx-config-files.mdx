# Understanding Nginx Configuration Files

## Main Configuration File
`/etc/nginx/nginx.conf` - The main configuration file

### Basic Structure
\`\`\`nginx
user nginx;
worker_processes auto;
error_log /var/log/nginx/error.log;
pid /var/run/nginx.pid;

events {
    worker_connections 1024;
}

http {
    include /etc/nginx/mime.types;
    default_type application/octet-stream;

    access_log /var/log/nginx/access.log;

    include /etc/nginx/conf.d/*.conf;
    include /etc/nginx/sites-enabled/*;
}
\`\`\`

## Configuration Contexts

### Main Context (top level)
Process-level settings
\`\`\`nginx
user nginx;
worker_processes auto;
error_log /var/log/nginx/error.log;
\`\`\`

### Events Context
Connection processing
\`\`\`nginx
events {
    worker_connections 1024;
    use epoll;
}
\`\`\`

### HTTP Context
Web server settings
\`\`\`nginx
http {
    include mime.types;
    sendfile on;
    keepalive_timeout 65;
}
\`\`\`

### Server Context
Virtual host settings
\`\`\`nginx
server {
    listen 80;
    server_name example.com;
    root /var/www/example;
}
\`\`\`

### Location Context
URL routing
\`\`\`nginx
location / {
    try_files $uri $uri/ =404;
}
\`\`\`

## Include Files
\`\`\`nginx
include /etc/nginx/mime.types;
include /etc/nginx/conf.d/*.conf;
include /etc/nginx/sites-enabled/*;
\`\`\`

## Configuration Files Layout
```
/etc/nginx/
├── nginx.conf              # Main config
├── mime.types              # MIME types
├── conf.d/                 # Additional configs
├── sites-available/        # All site configs
├── sites-enabled/          # Active sites (symlinks)
└── snippets/               # Reusable config snippets
```

You now understand Nginx config structure!
