# Nginx Configuration Files

## Main Config

The file `/etc/nginx/nginx.conf` contains:

```nginx
user nginx;
worker_processes auto;
error_log /var/log/nginx/error.log;

events {
    worker_connections 1024;
}

http {
    server {
        listen 80;
        server_name example.com;
    }
}
```

## Contexts

- Main
- Events
- HTTP  
- Server
- Location
