# Nginx Configuration Files

The main config file is `/etc/nginx/nginx.conf`

## Basic Structure

```nginx
user nginx;
worker_processes auto;

events {
    worker_connections 1024;
}

http {
    server {
        listen 80;
        server_name example.com;
    }
}
```

Done!
