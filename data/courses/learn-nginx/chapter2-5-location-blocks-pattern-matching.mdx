# Location Blocks and Pattern Matching

## Basic Location

\`\`\`nginx
location / {
    root /var/www;
}
\`\`\`

## Modifiers

| Modifier | Match Type |
|----------|------------|
| `=` | Exact |
| `^~` | Prefix (priority) |
| `~` | Regex (case-sensitive) |
| `~*` | Regex (case-insensitive) |

## Examples

\`\`\`nginx
# Exact match
location = /api {
    return 200 "API";
}

# Regex - PHP files
location ~ \\.php$ {
    fastcgi_pass php:9000;
}

# Images
location ~* \\.(jpg|png)$ {
    expires 30d;
}

# Catch-all
location / {
    try_files $uri =404;
}
\`\`\`
