# Prometheus and Grafana

## Nginx Prometheus Exporter

```bash
docker run -p 9113:9113 nginx/nginx-prometheus-exporter:latest \
  -nginx.scrape-uri http://localhost/nginx_status
```

## Prometheus Config

```yaml
scrape_configs:
  - job_name: nginx
    static_configs:
      - targets: ['localhost:9113']
```
