# Directory Structure

## Layout

```
/etc/nginx/
├── nginx.conf
├── sites-available/
├── sites-enabled/
└── conf.d/
```

## Best Practices

- Use sites-available/sites-enabled
- Keep separate log files
- Organize by domain
