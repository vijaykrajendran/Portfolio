# ConfigMaps and Secrets

## ConfigMap for Config

```yaml
apiVersion: v1
kind: ConfigMap
metadata:
  name: nginx-config
data:
  nginx.conf: |
    user nginx;
    worker_processes auto;

    http {
      server {
        listen 80;
      }
    }
```

## Secrets for Certificates

```yaml
apiVersion: v1
kind: Secret
metadata:
  name: nginx-tls
type: kubernetes.io/tls
data:
  tls.crt: base64_encoded_cert
  tls.key: base64_encoded_key
```

K8s config managed!
