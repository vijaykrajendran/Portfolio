# Index Files and Try Files

## Index Directive

\`\`\`nginx
server {
    index index.html index.htm;
}
\`\`\`

## Try Files

\`\`\`nginx
location / {
    try_files $uri $uri/ /index.html =404;
}
\`\`\`

## SPA Setup

\`\`\`nginx
server {
    root /var/www/app;

    location / {
        try_files $uri /index.html;
    }
}
\`\`\`

Perfect for single-page apps!
