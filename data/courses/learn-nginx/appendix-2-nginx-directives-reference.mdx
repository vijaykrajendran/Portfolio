# Nginx Directives Reference

## Common Directives

| Directive | Purpose |
|-----------|---------|
| `listen` | Port to listen on |
| `server_name` | Domain name |
| `root` | Document root |
| `location` | URL routing |
| `proxy_pass` | Forward request |
| `return` | HTTP redirect |
| `rewrite` | URL rewrite |
| `add_header` | Add header |
| `expires` | Cache expiry |
| `gzip` | Enable compression |

Quick reference!
