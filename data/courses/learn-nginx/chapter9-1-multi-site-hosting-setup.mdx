# Multi-Site Hosting Setup

## Directory Structure

```
/etc/nginx/sites-available/
├── blog.com
├── api.com
└── admin.com

/var/www/
├── blog.com/
├── api.com/
└── admin.com/
```

## Site Configuration

```nginx
server {
    listen 80;
    server_name blog.com www.blog.com;
    root /var/www/blog.com;

    access_log /var/log/nginx/blog.com.access.log;
    error_log /var/log/nginx/blog.com.error.log;
}
```

Multiple sites served!
