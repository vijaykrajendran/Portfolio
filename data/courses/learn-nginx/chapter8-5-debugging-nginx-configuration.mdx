# Debugging Configuration

## Test Syntax

```bash
sudo nginx -t
```

## Show Full Config

```bash
sudo nginx -T
```

## Enable Debug Logs

```nginx
error_log /var/log/nginx/error.log debug;
```

## Check Directives

```bash
# List directive syntax
man nginx

# Verify specific config
grep -n "proxy_pass" /etc/nginx/nginx.conf
```

## Reload and Check

```bash
sudo nginx -t
sudo systemctl reload nginx
sudo systemctl status nginx
```
