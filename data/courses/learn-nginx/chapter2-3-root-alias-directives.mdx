# Root and Alias Directives

## Root Directive

\`\`\`nginx
server {
    root /var/www/example;

    location / {
        # /file.html → /var/www/example/file.html
    }
}
\`\`\`

## Alias Directive

\`\`\`nginx
server {
    location /downloads/ {
        alias /mnt/storage/files/;
        # /downloads/file.zip → /mnt/storage/files/file.zip
    }
}
\`\`\`

## Difference

| Feature | Root | Alias |
|---------|------|-------|
| Appends path | ✅ | ❌ |
| Use case | General | Special paths |

Choose wisely!
